Core/Helpers Module¶

PyFynance.core.helpers.find_all_files(path, patterns, recursive=False, full_path=True)[source]¶

This method will find all the file in a specified path that match one of the patterns provided. this method also allows you to define if it should look recursivly or only at the top level

Parameters

  • path – string, path to search for files

  • patterns – list, of patterns to search for

  • recursive – boolean, indicating if the search should be recursive. default value is False

  • full_path – boolean, indicates if the returned values should eb full paths or just file names. default value is True

Returns

list: either full file paths for just file names based on the value of full_path
