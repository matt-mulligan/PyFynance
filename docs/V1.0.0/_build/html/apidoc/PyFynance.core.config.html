Core/Config Module¶

class PyFynance.core.config.Configuration[source]¶

Bases: object

The configuration module handles the loading and management of all configuration code and values within PyFynance. all setup is run for this module off of the __init__ call, which will return you a fully configured python object that you can get configuration values off of using dot notation

config = Configuration()
input_path = config.paths.input_path

The setting of values on this configuration object are governed by the configuration marshmallow schemas found in the schemas.config module, as well as the configuration json from the resources/config module that holds all of the actual config values to be loaded.
