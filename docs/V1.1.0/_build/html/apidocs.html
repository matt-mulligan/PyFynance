PyFynance API Documentation¶

PyFynance is built on a Task/Runner architecture where a task represents an end-to-end function of PyFynance that end users will use directly. This section will provide the API level documentation for the moduels that make up the PyFynance application. It is expected that anyone who works on PyFynance will addequately update the module, class and method level docstrings so that the API documentation is maintainable with sphinx-apidoc
