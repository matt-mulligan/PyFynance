PyFynance.tasks.task_categorize_transactions module¶

class PyFynance.tasks.task_categorize_transactions.CategorizeTransactionsTask(args)[source]¶

Bases: tasks.task_base.BaseTask

The Categorize Transactions task is responsible loading transactions from the transactions database, categorising them according to the rules within the rules database then writing the results back to the transactions DB.

ChangeLog:
